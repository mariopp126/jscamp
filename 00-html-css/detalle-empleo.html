Desarrollador Frontend Senior

Tech Solutions S.A.

Madrid, España

Descripción del puesto

Buscamos un Desarrollador Frontend Senior con experiencia en React y Vue.js para unirse a nuestro equipo dinámico. El candidato ideal tendrá un fuerte enfoque en la experiencia del usuario y la optimización del rendimiento.

Responsabilidades

  • Desarrollar y mantener aplicaciones web responsivas.
  • Colaborar con diseñadores y desarrolladores backend.
  • Optimizar aplicaciones para máxima velocidad y escalabilidad.

Requisitos

  • 5+ años de experiencia en desarrollo frontend.
  • Dominio de React, Vue.js, HTML5, CSS3 y JavaScript ES6+.
  • Experiencia con herramientas de control de versiones (Git).
